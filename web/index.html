awesome-nostr-relays

A curated catalogue of Nostr relays, organised into discovery collections and refreshed regularly by CI. Source on GitHub.

Status

Loading live status…

Endpoints

  • urls.json — flat URL list
  • relays.json — full dataset with health, monitor & summary
  • collections.json — grouped by collection id
  • nip66.json — NIP-66 kind 30166 discovery events
  • health.json — raw health snapshot
  • badge.json — shields.io health badge endpoint
  • schema/ — JSON Schema for catalog entries

Served with Content-Type: application/json, permissive CORS, no rate limit. 

curl -sL https://relays.meowl.social/urls.json | jq .urls
