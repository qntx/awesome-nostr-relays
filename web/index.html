Awesome Nostr Relays

Curated Nostr relay catalog. This domain serves a static JSON API generated from qntx/awesome-nostr-relays and refreshed hourly by CI.

Endpoints

  • relays.json Full dataset with curator metadata and CI-measured health status.
  • urls.json Flat list of relay URLs — lightest format, ideal for client-side relay rotation.
  • collections.json Relays grouped by collection id (featured, global, …).

Usage

Proper Content-Type: application/json, permissive CORS, no rate limit.

curl -sL https://relays.meowl.social/urls.json | jq .urls
